# simplicity-assignment

Home assignment for Simplicity interview

## Assignment Description

Assignment is a React / NestJS application aimed at fullstack development showcase.

Shall be a simple CRU(D) app for announcements management (listing them, creating them and updating them) as per received assignment.
Implementation follows detailed requirements.

Visual implementation roughly follows provided screenshots as a reference of expectations

Techstack used:
- React
- NestJS
- PostgreSQL
- Docker
- Docker Compose
- Tanstack Query
- TailwindCSS
- Tanstack Router
- Shadcn/UI / Shadcn/create
- Zod
- React Hook Form
- Vite
- ESLint
- Prettier
- tRPC and tRPC Panel
